## Introduction

RockNext is an Open Source extending ERPNext built on top of Frappe bringing enterprise ready utilization.

## Installation

```bash
# get app
$ bench get-app https://github.com/matbrgz/rocknext

# install on site
$ bench --site <sitename> install-app rocknext
```
